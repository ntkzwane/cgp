Detailed Description

RAII wrapper around creation and destruction of a temporary directory.

On construction, the directory is first deleted if it exists, then created. On destruction, the directory is deleted.

Warning
If a relative directory name is used, do not change the current working directory during the lifetime of this object, as otherwise the wrong directory will be removed on destruction.
