T  b

Tests whether the ratio between two floating-point values is exactly a power of 2.

The return value is always false if either value is negative or nonfinite.

The case with 65536 is to reduce the number of recursive calls required, making it usable in a constexpr setting.

static constexpr int shrRound ( int  x,
int  shift 
)
inlinestatic

Divides a value by 2shift, rounding to nearest.

Ties are broken by rounding towards +inf.

Precondition
Assuming that int is a 32-bit type,
  • 0 <= shift < 32
  • -230 < x < 230 The answer will be right in more cases than this, but it is a conservative estimate.
Note
The implementation assumes a twos-complement machine with an arithmetic right shift.
